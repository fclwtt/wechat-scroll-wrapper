# 微信通讯录滚动工具

配合你的自动点击工具使用，实现联系人列表的精确滚动。

## 工作原理

```
你的点击工具（exe）          滚动脚本（Python）
     │                           │
     ├─ 选中联系人                │
     ├─ 发送文字/图片             │
     ├─ 点击悬浮按钮 ──────────→ 触发滚动
     ├─ 选中下一个联系人          │
     ├─ 发送文字/图片             │
     ├─ 点击悬浮按钮 ──────────→ 触发滚动
     └─ ...循环                  └─ ...
```

两个脚本完全独立，通过**屏幕坐标点击**通信。

## 文件说明

```
scroll_wrapper/
├── main.py          # 悬浮按钮（运行这个）
├── scroller.py      # 核心滚动逻辑
├── calibrate.py     # 校准工具（首次使用）
├── config.json      # 配置文件（校准后生成）
└── README.md        # 本文件
```

## 快速开始

### 1. 安装依赖

```bash
pip install pyautogui pillow
```

### 2. 首次校准

```bash
python calibrate.py
```

按提示操作：
- 设置目标坐标（联系人头像对齐位置）
- 设置滚动区域（鼠标滚轮操作位置）
- 测量 dy_same（同字母间距）
- 测量 dy_cross（跨字母间距）

### 3. 启动滚动按钮

```bash
python main.py
```

会在屏幕上显示一个绿色的"下一步"按钮，始终置顶。

### 4. 配置你的点击工具

在点击工具的循环流程中，发完消息后加一步：

```
点击坐标 (button_x, button_y)  ← 悬浮按钮的位置
```

## 校准技巧

### 如何准确测量间距

**dy_same（同字母间距）**：
1. 找到同一字母下的两个联系人（如 H 下的"豪哥2号"和"后来"）
2. 将鼠标移到第一个联系人头像中心
3. 按 Enter 记录位置
4. 手动滚动鼠标，让第二个联系人移到相同位置
5. 将鼠标移到第二个联系人头像中心
6. 按 Enter，脚本自动计算距离

**dy_cross（跨字母间距）**：
- 同理，但选择相邻字母组的联系人（如 A → B）

### 如果找不到同字母联系人

使用默认值：
- dy_same = 56
- dy_cross = 88

或者手动编辑 `config.json` 调整。

## 配置说明

`config.json` 关键字段：

```json
{
    "target_pos": [x, y],      // 联系人头像对齐目标坐标
    "scroll_pos": [x, y],      // 鼠标滚轮操作位置
    "dy_same": 56,             // 同字母间距（像素）
    "dy_cross": 88,            // 跨字母间距（像素）
    "button_x": 100,           // 悬浮按钮 X 坐标
    "button_y": 100,           // 悬浮按钮 Y 坐标
    "button_width": 120,       // 按钮宽度
    "button_height": 50        // 按钮高度
}
```

## 常见问题

**Q: 滚动不准怎么办？**
A: 重新运行 `calibrate.py`，更仔细地测量间距。

**Q: 点击工具点不到按钮？**
A: 调整 `config.json` 中的 `button_x`、`button_y`，或增大 `button_width`、`button_height`。

**Q: 如何暂停/恢复？**
A: 关闭悬浮按钮窗口即可暂停。重新运行 `main.py` 恢复。

**Q: 支持不同分辨率？**
A: 每次调整微信窗口大小后，需要重新校准。

## 快捷键

在悬浮按钮窗口激活时：
- `Space` = 手动触发滚动
- `Esc` = 退出

## 注意事项

- 微信窗口位置必须固定，不能拖动
- 校准后不要调整微信窗口大小
- 确保悬浮按钮不被其他窗口遮挡
- 滚动脚本需要在点击工具之前启动
